# ApplyBoard 求职申请管理平台

一个面向个人求职场景的申请流程管理平台 Demo，用于统一管理岗位投递进度、面试流程、关键提醒事项及简历版本信息。

## 网址

https://applyboard-ebon.vercel.app

---

## 项目简介

ApplyBoard 聚焦求职过程中的信息分散、进度跟踪困难、节点提醒缺失等问题，尝试通过轻量化产品设计，将申请管理、流程推进和信息组织整合到统一工作台中。

平台支持看板视图与列表视图两种模式，可用于管理从投递到 Offer 的完整流程。

覆盖流程阶段包括：

- 待投递  
- 已投递  
- 笔试中  
- 面试中  
- Offer中  
- 已结束

---

## 核心功能

### 1. 求职流程管理
- 多阶段申请进度管理  
- 看板式流程追踪  
- 流程节点可视化展示  
- 侧拉详情页状态更新

### 2. 优先处理提醒
- 截止日期提醒  
- 高优先级申请提示  
- 待处理事项合集

### 3. 邮件提醒模块
- 模拟面试/笔试通知提醒  
- 关键流程节点提醒

### 4. 简历版本管理
- 不同岗位方向简历版本管理  
- 使用频次记录  
- 最近使用记录展示

### 5. 真实岗位链接跳转
部分申请记录接入真实岗位链接，可直接跳转外部招聘页面。
